# 待测试

## 充值与商业化系统

- 充值套餐：支持创建一次性额度套餐（体验包/基础包/标准包/专业包），用户购买后自动增加额度。
- 订阅会员：支持月度/季度/年度订阅套餐，购买后创建订阅记录并发放当期额度，到期续费可延长有效期。
- 订单生命周期：完整状态机（PENDING → PAID → RECHARGING → COMPLETED），支持取消、超时过期、支付失败标记。
- 支付回调：EasyPay 和 Stripe 回调验签 + CAS 原子状态更新 + 金额校验 + 幂等充值。
- 退款流程：管理员可发起退款（先扣后退），支持网关退款失败时回滚用户额度。
- 兑换码：管理员可批量生成额度/订阅兑换码，用户通过兑换码原子领取（CAS 防并发）。
- 优惠码：注册时使用优惠码可获得赠送额度。
- 风控限额：三层限额（单笔/每日用户/每日渠道）+ 待支付订单上限 + 取消频率限制 + 订单超时。
- 支付商配置：管理后台可在线配置 EasyPay/Stripe 实例，配置加密存储（AES-256-GCM）。
- 收入仪表盘：今日/总收入、订单数、成功率、客单价、每日趋势、渠道分布、充值排行榜。
- 管理后台：订单管理（筛选/退款）、套餐管理（增删改）、兑换码管理（批量生成）。
- 定时任务：订单超时清理 + 订阅过期标记。

## 既有功能

- 配置与用户偏好：每个模型渠道新增 OpenAI / Gemini 调用格式选择，默认 OpenAI，切换默认格式时同步更新默认 Base URL；渠道 Tab 增加紧凑醒目的提醒，说明模型是否可选需要到"模型"Tab 选择。
- Gemini 调用格式：支持拉取 Gemini 模型列表，并用于文本问答、在线 Agent 工具调用、基础生图和图生图请求。
- 音频和视频生成：选择 Gemini 调用格式时先给出不支持提示，仍需使用 OpenAI 兼容渠道。
- 登录安全：生产环境缺少 `JWT_SECRET` 时不再回退到公开默认密钥，需确认部署环境已正确注入 `JWT_SECRET` 且登录态正常。
